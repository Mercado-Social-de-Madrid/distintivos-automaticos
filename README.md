# Distintivos Automáticos

## Instalación

```bash
$ python3.10 -m venv .venv
$ source .venv/bin/activate
(.venv) $ pip install -e .
```

## Ejecución

Hacen falta:

- Los logos
- Un archivo `.csv` que contenga nombre de la entidad y URL del logo
- Los directorios de destino apropiados creados
- Las plantillas

```bash
(.venv) $ bash main.sh
```
